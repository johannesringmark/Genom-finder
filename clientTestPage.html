SNPs/Genom Sandbox

This application aims to make it easier, when comparaing snps based gene results against the markers and trees on the cotained on the ISSOG home page.
example:
  • Choose File "genom_test.txt"
  • Choose File "test.db23"
  • Press "Find Mutations"
  • Save Mutations in a new document ex "test.mut23"
  • Choose File "test.mut23"
  • Choose File "ISSOG.tree23"
  • Press "Find Haplogroup"
  • Possible Haplogroups is listed..
